在這個範例中，我們藉由 <a href="#top"> 來動態地操控 id 是 top 的 HTML 元素樣式。

<a href="#top"> 對應的 target 就是 id 為 top 的 HTML 元素。

當我們點選其他連結的時候，連結對應 id 的元素也會改變樣式，同時其他元素的樣式會恢復原狀。
